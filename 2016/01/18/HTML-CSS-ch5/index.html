HTML+CSS 第五章 与浏览者交互，表单标签

5-1 <table>标签

  • 语法:

    <form method="传送方式" action="服务器文件">

  • 讲解:

    <form></from>成对出现

action: 浏览器输入数据被传送到的地方

method: 数据传送方式(post/get)

  • 注意:

    表单控件(文本框,文本域,按钮,单选框,复选框...)都必须在<form>之间

5-2文本框、密码输入框

  • 语法:

    <form>
    <input type="text/password" name="名称" value="文本"/>
</form>

  • 讲解

    type:

    当type="text"时,输入框为文本框
    当type="password"时,输入框为密码框

name:

    为文本框命名,以备后台程序ASP,PHP使用.

value:

    为文本输入框设置默认值.(一般起到提示作用)

5-3文本域,支持多行文本输入

  • 语法

    <textarea rows="行数" cols="列数">文本</textarea>

  • 讲解

    <textarea>:

    成对出现,以<textarea>开始，以</textarea>结束

cols:

    输入域的列数

rows:

    输入域的行数

默认值：

    <textarea></textarea>之间可以有默认值

  • 注意

    cols和rows可以用CSS样式的width和height来替代

5-4使用单选框,复选框让用户选择

  • 语法

    <input type="radio/checkbox" value="值" name="名称" checked="checked"/>

  • 讲解

    type:

    当type="radio"时,控件为单选框
    当type="checkbox"时,控件为复选框

value:

    提交数据到服务器的值(后台程序使用)

name:

    为控件命名,以备后台程序ASP,PHP使用

checked:

    当设置checked="checked"时，该选项被默认选中

  • 注意

    name：

    同一组的单选按钮name取值要一致，这样才能起到单选的效果

5-5使用下拉列表,节省空间

  • 语法

    <select>
    <option value="提交值" selected="selected">显示值</option>
    <option value="提交值">显示值</option>
    ....
</select>

  • 讲解

    设置selected="selected",则该选项被默认选中

5-6使用下拉列表框进行多选

  • 语法

    <select multiple="multiple">
    <option value="提交值" selected="selected">显示值</option>
    <option value="提交值">显示值</option>
    ....
</select>

5-7使用提交按钮,提交数据

  • 语法

    <input type="submit" value="提交">

  • 讲解

    type:

    当type="submit"时,才有提交功能

value:

    按钮上显示的文字

5-8使用重置按钮,重置表单信息

  • 语法

    <input type="reset" value="重置">

  • 讲解

    type:

    当type="reset"时,按钮用重置作用

value:

    按钮上显示的文字

5-9form表单中的label标签

  • 说明

    label标签不会向用户呈现任何特殊效果.
如果你在 label 标签内点击文本,就会触发此控件.
就是说,当用户单击选中该label标签时,
浏览器就会自动将焦点转到和标签相关的表单控件上
(就自动选中和该label标签相关连的表单控件上).

  • 语法

    <label for="控件id名称">

  • 注意

    标签for属性中的值应当与相关控件的id属性值一定要相同.
