HTML+CSS 第六章 开始学习CSS,为网页添加样式

6-1 认识CSS样式

  • 说明:

    CSS全程“层叠样式表（Cascading Style Sheets）”主要用于定义HTML内容在浏览器中的显示样式，如文字大小、颜色、字体、加粗等。
使用CSS样式的好处是通过定义某个样式，可以让不同网页位置的文字有着统一的字体、字号、颜色等。

  • 举例:

    p{
    font-size:12px;//设置字号
    color:red;//设置文字颜色
    font-weight:blod;//设置字体加粗
}

6-2 CSS样式的优势

  • 说明
    使用CSS，可以定义一种样式在多处使用，使界面统一、管理方便

6-3 CSS代码语法

  • 说明

    CSS语法由 **选择符** 和 **声明** 组成，声明又可以分为 **属性** 和 **值** 。属性和值由 : 分隔，选择符用 ; 分隔。

  • 举例

    p{
    font-size:12px;
    color:red;
}
/*
    p 是声明，HTML中被声明包裹的字段都将应用此声明的样式

    font-size、color是属性，color、red分别为属性对应的值
*/

6-4 CSS代码注释

  • 语法

    /*注释文字*/
p
{
    font-size:12px;
    color:red;
}
