第6章 开始学习CSS,为网页添加样式

6-1 认识CSS样式

  • 说明:

    CSS全程“**层叠样式表**（Cascading Style Sheets）”主要用于定义HTML内容在浏览器中的显示样式，如文字大小、颜色、字体、加粗等。
使用CSS样式的好处是通过定义某个样式，可以让不同网页位置的文字有着统一的字体、字号、颜色等。

  • 举例:

    p{
    font-size:12px;//设置字号
    color:red;//设置文字颜色
    font-weight:blod;//设置字体加粗
}
